How to use this page:

This page automatically receives Twitch events and dispatches them as DOM events. Listen for them like this:

document.addEventListener('twitch:raid', (event) => {
  const { displayName, viewerCount } = event.detail
  console.log(`${viewerCount} viewers raided from ${displayName}!`)
  // Show your custom notification here!
})

Available events: twitch:raid, twitch:follow, twitch:subscribe, twitch:gift, twitch:cheer, twitch:redemption
